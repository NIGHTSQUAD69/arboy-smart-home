# Arboy Smart Home

Welcome to Arboy Smart Home — a personal smart home automation system designed to integrate seamlessly with Google Home and modern IoT devices.

## About

Arboy Smart Home is built to provide efficient, secure, and flexible control of home devices such as lighting, sensors, and automation systems.

This project focuses on:

* Smart automation
* Device control
* Integration with Google Home
* Reliable and secure operations

## Features

* Real-time device control
* Automation triggers
* Integration with smart home platforms
* Lightweight and efficient system
