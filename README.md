# Arboy Smart Home

Arboy Smart Home is a personal smart home automation system designed to integrate with platforms such as Google Home.

## Overview

This project enables control and automation of home devices including lighting, sensors, and other IoT components.

## Features

* Smart device control
* Automation triggers
* Integration with Google Home
* Lightweight and efficient system

## Project Scope

This system is intended for personal smart home automation and small-scale deployments.
